# HTML-AND-CSS-PROJECTS

This repository is for projects created for The Tech Academy's HTML and CSS course. All the "pages" projects use HTML and CSS, and some use Bootstrap to style the pages. 

## Projects

- Academy Cinemas Page
- Photo Album Page 
- Challenges 1-5 Page
- Grace Hopper Page
- Portfolio Page 
- Simple Recipes Page
- The Pet Shop Page
- Small Assignments Folder

### Academy Cinemas Page
This page is an example home page for an imaginary movie theater styled using Bootstrap components. 

### Photo Album Page
This is a one-page photo album created using a Bootstrap example template. 

### Challenges 1-5 Page
This is a simple page showcasing a particular photograph. It might be the type of page a photographer would have as part of their website.

### Grace Hopper Page
This is a page giving biographical information about the late, great computer scientist Grace Hopper.

### Portfolio Page
This is a page I made to showcase the major assignments I did in this course. It features responsive sizing that I figured out how to do on my own using online resources like w3schools.com

### Simple Recipes Page
This is a page offering, as the title suggests, simple recipes of the type a food blogger might create. 

### Small Assignments Folder
This is a catch-all folder for the various minor assignments completed during the course.

### The Pet Shop Page
This is an example page for an imaginary small pet shop.
